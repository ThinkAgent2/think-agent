### 1. Le Conseiller McKinsey

| Attribut | Valeur |
|----------|--------|
| **Description** | Créer un assistant de conseil stratégique maîtrisant les frameworks McKinsey : MECE, Pyramide de Minto, Issue Trees, Hypothesis-Driven Thinking. L'assistant challenge les idées, structure les raisonnements et utilise la maïeutique socratique pour faire émerger les angles morts. |
| **Difficulté** | ⭐⭐⭐ |
| **Durée** | 2h |
| **Marque** | Tous |
| **Participants** | Solo |
| **Livrables** | Assistant fonctionnel + session de travail documentée sur un cas réel |
| **Outils** | Chat IA |
| **Évaluation** | Peer review |
| **Critères** | Maîtrise des frameworks, qualité du questionnement, pertinence des challenges |
| **XP** | 150 |
